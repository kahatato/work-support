# Git-bashとは  
Windows上で、Gitやbash(linux)の操作ができるターミナル。  
# 基本的な操作  
## ターミナル切り替え  
### powershellに切り替え  
~~~
powershell
~~~
### コマンドプロンプトに切り替え  
~~~
cmd
~~~
### 切り替えた後、抜け出す  
~~~
exit
~~~
### Git-GUI起動
~~~
git-gui
~~~
### Git-GUI終了
Git-GUIのウィンドウを閉じる

### Vim起動
~~~
vim
~~~
### コマンド実行終了
~~~
:q
~~~
